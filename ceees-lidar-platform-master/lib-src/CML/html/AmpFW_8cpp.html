Detailed Description

This file contains code used to update an amplifier's firmware over the CANopen network.

The firmware update protocol is a non-standard protocol which uses CAN messages with the same IDs as the node's CANopen SDO protocol. These message IDs were used because they are fixed for a given node ID.
